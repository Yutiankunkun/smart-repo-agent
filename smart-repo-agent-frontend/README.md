# Smart Repo Agent Frontend

A practice project — React + Tailwind CSS frontend for student management and AI meeting assistant.

> **Note:** This frontend was built with [vibe coding](https://cursor.com) (AI-assisted development).

## Tech Stack

- React 18+
- Tailwind CSS
- React Router
- Axios

## Development

```bash
npm install
npm run dev
```

Runs at http://localhost:5173 by default.

## Build

```bash
npm run build
```

## Environment

Backend API defaults to `http://localhost:8000`. Ensure the backend is running.

## Features

- **Student Management**: CRUD for students, teacher filter
- **Teacher Management**: CRUD for teachers (separate tab)
- **AI Chat**: Conversation with AI meeting assistant, with collapsible tool execution display
